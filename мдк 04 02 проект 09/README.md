# Система аутентификации по номеру телефона

Современная система аутентификации на PHP с использованием SQLite для портфолио. Включает регистрацию, вход, восстановление пароля через секретный вопрос и защиту от основных уязвимостей.

## 🚀 Возможности

- ✅ Регистрация пользователей по номеру телефона
- ✅ Безопасный вход в систему
- ✅ Восстановление пароля через секретный вопрос
- ✅ CSRF-защита
- ✅ Хэширование паролей (bcrypt)
- ✅ Валидация данных на сервере
- ✅ Адаптивный современный дизайн
- ✅ Подготовленные SQL-запросы (защита от SQL-инъекций)

## 📋 Требования

- PHP 8.0 или выше
- SQLite3 (обычно встроен в PHP)
- Веб-сервер (Apache, Nginx или встроенный PHP сервер)

## 🛠️ Установка

1. **Клонируйте репозиторий или скопируйте файлы проекта**

2. **Инициализируйте базу данных:**
   ```bash
   php backend/init.php
   ```
   Это создаст файл `backend/database.db` с необходимой структурой таблиц.

3. **Настройте веб-сервер:**
   
   **Вариант 1: Встроенный PHP сервер (для разработки)**
   ```bash
   php -S localhost:8000
   ```
   Затем откройте в браузере: `http://localhost:8000/backend/register.php`

   **Вариант 2: Apache/Nginx**
   - Настройте виртуальный хост, указывающий на корневую директорию проекта
   - Убедитесь, что PHP имеет права на запись в папку `backend/` (для создания БД)

## 🗄️ Структура базы данных

### Таблица `users`

| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER PRIMARY KEY | Уникальный идентификатор |
| phone | TEXT UNIQUE | Номер телефона (формат: +7-XXX-XXX-XX-XX) |
| password | TEXT | Хэшированный пароль |
| secret_question | TEXT | Ключ секретного вопроса |
| secret_answer | TEXT | Хэшированный ответ |
| created_at | DATETIME | Дата регистрации |

## 🔐 Безопасность

- **Хэширование паролей**: Используется `password_hash()` с алгоритмом bcrypt
- **CSRF-защита**: Все формы защищены токенами
- **Подготовленные запросы**: PDO с prepared statements для защиты от SQL-инъекций
- **Валидация данных**: Проверка на сервере всех входящих данных
- **Безопасные сессии**: Настроены параметры безопасности сессий
- **Хэширование ответов**: Ответы на секретные вопросы также хэшируются

## 📝 Использование

### Регистрация

1. Перейдите на `/backend/register.php`
2. Введите номер телефона в формате `+7-XXX-XXX-XX-XX`
3. Создайте пароль (минимум 8 символов, буквы и цифры)
4. Выберите секретный вопрос и укажите ответ
5. Нажмите "Зарегистрироваться"

### Вход

1. Перейдите на `/backend/login.php`
2. Введите номер телефона и пароль
3. Нажмите "Войти"

### Восстановление пароля

1. На странице входа нажмите "Забыли пароль?"
2. Введите номер телефона
3. Ответьте на секретный вопрос
4. Установите новый пароль

## 🔧 Настройка

### Изменение формата телефона

Отредактируйте функцию `validatePhone()` в `backend/config.php` для изменения формата.

### Добавление секретных вопросов

Добавьте вопросы в функцию `getSecretQuestions()` в `backend/config.php`.

### Настройка безопасности сессий

В `backend/config.php` измените параметры:
```php
ini_set('session.cookie_secure', 1); // Для HTTPS
```

## 🐛 Обработка ошибок

Все ошибки логируются в системный лог PHP. Для просмотра ошибок в режиме разработки включите отображение ошибок в `php.ini`:

```ini
display_errors = On
error_reporting = E_ALL
```
